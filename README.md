# Desafio Perda de Peso - 4% Challenge

Um Web App moderno e elegante para ajudar colegas de trabalho a monitorarem sua meta de perda de peso de 4% em um mês.

## ✨ Funcionalidades

- **Cadastro e Login**: Cadastro simples com nome, e-mail e peso inicial.
- **Cálculo Automático de Meta**: O app calcula automaticamente a meta de -4% baseada no peso inicial.
- **Registro Diário**: Interface intuitiva para registrar o peso do dia.
- **Gráfico de Evolução Pessoal**: Acompanhamento visual da perda de peso ao longo do tempo.
- **Ranking Comparativo**: Gráfico que compara o progresso percentual de todos os participantes cadastrados (ideal para manter a motivação!).
- **Feedback Visual**: Mensagens de parabéns e progresso em tempo real.
- **Design Premium**: Interface escura com efeito Glassmorphism, gradientes e animações suaves.

## 🚀 Como Executar

Como o app foi desenvolvido com tecnologias web puras (HTML, CSS e JS), você pode executá-lo de duas formas:

1. **Localmente**: Basta abrir o arquivo `index.html` em qualquer navegador moderno.
2. **Servidor Local (Recomendado)**: Se você tiver o VS Code instalado, use a extensão "Live Server" para uma experiência melhor.

## 🛠️ Tecnologias Utilizadas

- **HTML5 / CSS3** (Vanilla com Flexbox/Grid)
- **JavaScript ES6+**
- **Chart.js**: Para visualização de dados.
- **Lucide Icons**: Para iconografia moderna.
- **LocalStorage**: Para persistência de dados no navegador.
